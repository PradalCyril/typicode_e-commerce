# GET STARTED 
Clone the folder
```
git clone https://github.com/PradalCyril/typicode_e-commerce.git
```

Change branch master to dev 
```
git checkout dev
```

Install the dependencies 
```
npm i 
npm install -g json-server
cd db/
json-server --watch db.json
```

Open new terminal and 
```
npm start
```
